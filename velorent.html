VeloRent – Car Rental System

Secure rental management platform with 2FA and RBAC controls.

Built a full-stack rental system with secure authentication, role management, and automated rental workflows.

  • Multi-factor authentication with secure session handling.
  • RBAC for admin, staff, and customer permissions.
  • Automated status tracking and vehicle availability updates.
Java Spring Boot SQL Server 2FA RBAC
GitHub

Overview

Web-based car rental platform built for the SE2030 Software Engineering module to replace manual tracking with a modern, reliable system.

Highlights

  • Vehicle discovery with live availability and quick booking.
  • Role-based access for customers, fleet managers, owners, and admins.
  • Secure payments with receipts and rental history tracking.
  • Fleet lifecycle dashboard and business reports.

Processes

  • Search vehicles, review details, and create a booking.
  • Pay by cash or card and receive a receipt.
  • Pickup, return, and close out with inspection and final invoice.
  • Package bookings support add-ons and duration-based pricing.

Roles

  • Owner: portfolio performance, vehicles, payouts, and escalations.
  • Fleet Manager: availability, maintenance, and booking oversight.
  • Administrator: accounts, security, configuration, and audits.

Tech Stack

  • Java 17, Spring Boot 3.5.5, Maven.
  • Thymeleaf, Bootstrap, JavaScript.
  • Microsoft SQL Server, RESTful services.
  • SQL scripts under src/main/resources/db.
