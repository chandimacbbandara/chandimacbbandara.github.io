AI-Powered Resume Screening System

Intelligent RAG-based recruitment tool that automates candidate evaluation with LLMs.

An intelligent, RAG-based recruitment tool that automates the candidate evaluation process. This system extracts text from uploaded PDF resumes, actively scrapes embedded portfolio links (like GitHub), and leverages an instruct-tuned Large Language Model to score the candidate against specific job requirements — all through a polished, interactive web dashboard.

  • Intelligent PDF parsing with pdfplumber for complex resume layouts.
  • Live portfolio scraping from GitHub and embedded links.
  • LLM-based scoring against job description requirements.
  • RAG pipeline with FAISS vector store for context retrieval.
  • Interactive web dashboard built with Gradio.

Overview

The AI-Powered Resume Screening System is built on a Retrieval-Augmented Generation (RAG) architecture. It ingests candidate PDF resumes, extracts and enriches the data by scraping linked portfolios (GitHub repositories, personal sites), and then passes all content through an instruct-tuned LLM that evaluates and scores the candidate against a given job description. The entire pipeline runs behind an interactive Gradio dashboard, deployable instantly on Hugging Face Spaces.

Core Features

  • Intelligent PDF Parsing: Accurately extracts complex layouts and text from candidate CVs using pdfplumber.
  • Live Portfolio Scraping: Detects and actively scrapes GitHub profile links and repository READMEs embedded in the resume.
  • RAG Pipeline: Combines parsed resume text with scraped portfolio data into a FAISS vector store for semantic retrieval.
  • LLM Scoring Engine: Uses an instruct-tuned LLM (via LangChain) to reason about the candidate's suitability and generate a structured score.
  • Interactive Dashboard: Gradio-based web UI for uploading resumes, entering job descriptions, and viewing detailed evaluation reports in real time.
  • One-Click Deployment: Fully containerized and deployable on Hugging Face Spaces with no infrastructure setup required.

System Architecture

The pipeline follows a modular RAG design:

  • Input Layer: PDF upload via Gradio UI + Job Description text input.
  • Extraction: pdfplumber parses resume text; BeautifulSoup / requests scrapes portfolio links.
  • Indexing: All text chunks are embedded and stored in a FAISS vector index.
  • Retrieval & Generation: LangChain retrieves relevant chunks and passes them to the LLM with a structured prompt.
  • Output: A detailed candidate score report with strengths, gaps, and an overall match percentage.

Technology Stack

  • Language: Python 3.10+
  • LLM Framework: LangChain (instruct-tuned model via HuggingFace Hub)
  • Vector Store: FAISS (Facebook AI Similarity Search)
  • PDF Parsing: pdfplumber
  • Web Scraping: requests + BeautifulSoup4
  • UI / Dashboard: Gradio
  • Deployment: Hugging Face Spaces
  • Embeddings: sentence-transformers (all-MiniLM-L6-v2)
