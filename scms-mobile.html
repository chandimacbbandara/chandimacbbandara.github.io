Student Concern Management System (SCMS) – Mobile

Mobile app for tracking student concerns with fast, secure access.

Designed a mobile-first experience so students can submit, update, and monitor concerns without being tied to a desktop interface.

  • JWT-secured authentication and session management.
  • Real-time status updates and push-ready notifications.
  • Clean, accessible UX for quick concern submission.
React Native Expo Node.js MongoDB JWT
GitHub

Overview

Multi-role mobile ecosystem built with the MERN stack to streamline student concerns, medical triage, and administration workflows.

Highlights

  • Secure authentication, concern submission, and real-time tracking.
  • Medical report uploads and consulter review workflows.
  • Admin dashboard for triage, replies, and resolution management.
  • Broadcast notices, analytics, and reporting for executives.

Tech Stack

  • React Native, Expo, React Navigation, Axios, Ionicons.
  • Node.js, Express.js, MongoDB (Atlas), Mongoose.
  • JWT, Bcryptjs, Multer, Nodemailer.
  • Custom glassmorphism UI and fluid animations.
