Student Mental Health Early Detection System

Machine learning model for proactive student wellbeing support.

This project builds an end-to-end pipeline to detect early signs of mental health risk in students using academic and stress-related factors. It includes preprocessing, exploratory analysis, and multiple machine learning models.

  • TensorFlow MLP model tuned for high recall.
  • Feature engineering from survey and behavioral data.
  • Clear evaluation metrics for model transparency.
Python TensorFlow MLP Keras Scikit-learn
GitHub

Focus

Early signs of mental health challenges are often subtle. This project aims to flag at-risk students sooner so support can be provided before issues escalate.

Impact

By analyzing attendance, grades, and stress factors, the system improves early intervention, reduces dropouts, and supports academic performance.
